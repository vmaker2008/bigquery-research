## BigQuery란?

**Google BigQuery**는 Google Cloud에서 제공하는 **완전 관리형 서버리스 엔터프라이즈 데이터 웨어하우스**입니다.

머신러닝, 지리정보 분석, 비즈니스 인텔리전스 등의 기능이 기본 내장되어 있으며, 별도의 인프라 관리 없이 페타바이트 규모의 데이터를 분석할 수 있습니다.

---

## 핵심 특징

### 1. 서버리스 (Serverless)
- 서버나 데이터베이스 관리자가 필요 없습니다.
- 인프라 프로비저닝, 패치, 튜닝을 Google이 자동으로 처리합니다.
- 사용자는 데이터 분석에만 집중할 수 있습니다.

### 2. 탁월한 성능
- **테라바이트 규모** 쿼리: **초(seconds)** 단위 처리
- **페타바이트 규모** 쿼리: **분(minutes)** 단위 처리
- 수천 개의 CPU 코어에 병렬로 쿼리를 분산 실행합니다.

### 3. 무한 확장성
- 데이터 크기에 따라 자동으로 확장됩니다.
- 스토리지와 컴퓨팅이 분리되어 있어 각각 독립적으로 확장 가능합니다.
- 수십 명에서 수천 명의 동시 사용자를 지원합니다.

### 4. AI/ML 통합
- **BigQuery ML**: SQL 쿼리만으로 머신러닝 모델 생성, 훈련, 예측
- **Vertex AI 통합**: 고급 AI 모델과 연동
- **생성 AI**: 텍스트 요약, 감정 분석, 기계 번역 등

### 5. 유연한 데이터 처리
- 정형(Structured) 및 반정형(Semi-structured), 비정형(Unstructured) 데이터 처리
- **개방형 형식 지원**: Apache Iceberg, Delta Lake, Apache Hudi
- JSON, Avro, Parquet, ORC 등 다양한 파일 포맷 지원

### 6. 엔터프라이즈 거버넌스
- **Dataplex Universal Catalog**: 데이터 자산 자동 추적 및 관리
- 데이터 계보(Data Lineage) 추적
- 자동 데이터 프로파일링 및 품질 점수

## BigQuery를 사용하는 이유

```
전통적 데이터웨어하우스          BigQuery
━━━━━━━━━━━━━━━━━━          ━━━━━━━━
서버 구매 및 설치        →    즉시 사용 가능
DBA 필요              →    NoOps, 자동 관리
수동 확장             →    자동 확장
행 기반 스토리지        →    열 기반 (분석 최적화)
높은 초기 비용         →    사용량 기반 과금
```

## 주요 활용 사례

| 산업 | 활용 사례 |
|------|----------|
| 커머스/리테일 | 판매 분석, 재고 예측, 고객 세분화 |
| 금융 | 리스크 분석, 사기 탐지, 실시간 거래 모니터링 |
| 의료 | 임상 데이터 분석, 환자 예측 모델 |
| 미디어 | 시청자 행동 분석, 콘텐츠 추천 |
| 게임 | 플레이어 행동 분석, 게임 밸런싱 |
| 마케팅 | 광고 효과 측정, A/B 테스트 분석 |
